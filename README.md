# ts-run Monorepo

Monorepo for ts-run and related packages.

## Packages

- [`ts-run`](./packages/ts-run) - Execute TypeScript/JavaScript with the runtime relative to your package manager

## Development

```bash
# Install dependencies
pnpm install

# Build all packages
pnpm build

# Run checks
pnpm check

# Format code
pnpm format
```

## Requirements

- **Node.js**: ≥22.6.0
- **pnpm**: ≥9.0.0
